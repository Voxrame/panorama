## Installation
*We actively develop a new version, which will be available soon.*

See the [**latest stable version: 0.0.3**](https://github.com/Voxrame/panorama/tree/stable-0.0.3?tab=readme-ov-file#panorama)

## Upcoming Features

- **Modes:** isometric 3D, top-down 2D
- **Database Support:** PostgreSQL, SQLite
- **UI:**
    - **Multiple layers**
    - **Zooming**
    - **Switching between modes**
    - **Search**
    - **Points of Interest:** markers and labels for important locations
- **Game-like view:** you see the world as in the game
